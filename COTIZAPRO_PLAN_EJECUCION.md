## 1. Resumen Ejecutivo

### 🎯 Visión
**"El Stripe para técnicos de mantenimiento"** - Profesionalizar servicios informales en 60 segundos.

### 💡 Problema
Técnicos de minisplit, plomería, electricidad y pintura pierden 40% de clientes potenciales por:
- ✗ Cotizaciones informales (WhatsApp, papel)
- ✗ Sin seguimiento de clientes
- ✗ Olvidan mantenimientos anuales
- ✗ Sin historial organizado

### ✅ Solución
Plataforma web/móvil que genera cotizaciones profesionales en 30 segundos con:
- ✓ Catálogo pre-cargado de servicios y precios
- ✓ PDF profesional con logo personalizado
- ✓ CRM automático con historial
- ✓ Recordatorios WhatsApp automatizados
- ✓ Calendario de servicios
- ✓ Cobro con link de pago (Stripe)
- ✓ Facturación CFDI 4.0

### 📊 Mercado Objetivo
- **TAM**: 2M técnicos × $2,400/año = **$4.8B MXN**
- **SAM**: 300K técnicos minisplit × $2,400/año = **$720M MXN**
- **SOM (Año 1)**: 1,000 técnicos × $2,400/año = **$2.4M MXN**

### 👥 Equipo Inicial
- **Tú**: Product + Development
- **Mes 4**: Contratar 1 vendedor
- **Mes 6**: Contratar 1 customer success

### 💰 Inversión Requerida
- **Capital inicial**: $100K-150K MXN
- **Tiempo a cashflow positivo**: 6-9 meses
- **Revenue Año 1**: $955K MXN ARR

### Arquitectura de Base de Datos

**Archivo**: `supabase/migrations/002_cotizapro_schema.sql`

```sql
-- ============================================
-- COTIZAPRO DATABASE SCHEMA
-- ============================================

-- Servicios/Productos (Catálogo)
CREATE TABLE services (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  organization_id UUID NOT NULL REFERENCES organizations(id) ON DELETE CASCADE,
  name TEXT NOT NULL, -- "Instalación Minisplit 1 Ton"
  description TEXT,
  category TEXT NOT NULL, -- "instalacion", "mantenimiento", "reparacion"
  base_price DECIMAL(10,2) NOT NULL,
  unit TEXT DEFAULT 'servicio', -- "servicio", "hora", "m2"
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Clientes
CREATE TABLE customers (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  organization_id UUID NOT NULL REFERENCES organizations(id) ON DELETE CASCADE,
  name TEXT NOT NULL,
  email TEXT,
  phone TEXT NOT NULL, -- REQUERIDO para WhatsApp
  address TEXT,
  city TEXT,
  notes TEXT,
  created_by UUID REFERENCES auth.users(id),
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Cotizaciones
CREATE TABLE quotes (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  organization_id UUID NOT NULL REFERENCES organizations(id) ON DELETE CASCADE,
  customer_id UUID NOT NULL REFERENCES customers(id) ON DELETE CASCADE,
  quote_number TEXT UNIQUE NOT NULL, -- "COT-2026-001"
  status TEXT DEFAULT 'draft' CHECK (status IN ('draft', 'sent', 'accepted', 'rejected', 'expired')),
  subtotal DECIMAL(10,2) NOT NULL,
  tax DECIMAL(10,2) DEFAULT 0, -- IVA 16%
  discount DECIMAL(10,2) DEFAULT 0,
  total DECIMAL(10,2) NOT NULL,
  valid_until DATE, -- Vigencia 15 días
  notes TEXT,
  pdf_url TEXT,
  sent_at TIMESTAMPTZ,
  accepted_at TIMESTAMPTZ,
  created_by UUID REFERENCES auth.users(id),
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Items de Cotización
CREATE TABLE quote_items (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  quote_id UUID NOT NULL REFERENCES quotes(id) ON DELETE CASCADE,
  service_id UUID REFERENCES services(id),
  description TEXT NOT NULL,
  quantity DECIMAL(10,2) NOT NULL DEFAULT 1,
  unit_price DECIMAL(10,2) NOT NULL,
  total DECIMAL(10,2) NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Citas/Servicios Programados
CREATE TABLE appointments (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  organization_id UUID NOT NULL REFERENCES organizations(id) ON DELETE CASCADE,
  customer_id UUID NOT NULL REFERENCES customers(id) ON DELETE CASCADE,
  quote_id UUID REFERENCES quotes(id),
  title TEXT NOT NULL,
  description TEXT,
  service_type TEXT NOT NULL,
  scheduled_at TIMESTAMPTZ NOT NULL,
  duration_minutes INTEGER DEFAULT 120,
  status TEXT DEFAULT 'scheduled' CHECK (status IN ('scheduled', 'confirmed', 'in_progress', 'completed', 'cancelled')),
  reminder_sent BOOLEAN DEFAULT false,
  notes TEXT,
  assigned_to UUID REFERENCES auth.users(id),
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Recordatorios Automáticos
CREATE TABLE automated_reminders (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  organization_id UUID NOT NULL REFERENCES organizations(id) ON DELETE CASCADE,
  customer_id UUID NOT NULL REFERENCES customers(id) ON DELETE CASCADE,
  reminder_type TEXT NOT NULL, -- "mantenimiento_anual", "seguimiento_cotizacion"
  next_reminder_date DATE NOT NULL,
  frequency_months INTEGER, -- 12 anual, 6 semestral
  last_service_date DATE,
  message_template TEXT,
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Índices para Performance
CREATE INDEX idx_services_org_id ON services(organization_id);
CREATE INDEX idx_customers_org_id ON customers(organization_id);
CREATE INDEX idx_customers_phone ON customers(phone);
CREATE INDEX idx_quotes_org_id ON quotes(organization_id);
CREATE INDEX idx_quotes_customer_id ON quotes(customer_id);
CREATE INDEX idx_quotes_status ON quotes(status);
CREATE INDEX idx_appointments_scheduled_at ON appointments(scheduled_at);
CREATE INDEX idx_reminders_next_date ON automated_reminders(next_reminder_date);

-- Row-Level Security (RLS)
ALTER TABLE services ENABLE ROW LEVEL SECURITY;
ALTER TABLE customers ENABLE ROW LEVEL SECURITY;
ALTER TABLE quotes ENABLE ROW LEVEL SECURITY;
ALTER TABLE quote_items ENABLE ROW LEVEL SECURITY;
ALTER TABLE appointments ENABLE ROW LEVEL SECURITY;
ALTER TABLE automated_reminders ENABLE ROW LEVEL SECURITY;

-- Políticas RLS (ejemplo para services, replicar para todas)
CREATE POLICY "Users view services in their org"
  ON services FOR SELECT
  USING (organization_id IN (
    SELECT organization_id FROM profiles WHERE id = auth.uid()
  ));

CREATE POLICY "Members create services"
  ON services FOR INSERT
  WITH CHECK (organization_id IN (
    SELECT organization_id FROM profiles WHERE id = auth.uid()
  ));

-- Función para generar número de cotización
CREATE OR REPLACE FUNCTION generate_quote_number(org_id UUID)
RETURNS TEXT AS $$
DECLARE
  year_part TEXT;
  count_part TEXT;
  new_number TEXT;
BEGIN
  year_part := TO_CHAR(NOW(), 'YYYY');

  SELECT LPAD((COUNT(*) + 1)::TEXT, 4, '0') INTO count_part
  FROM quotes
  WHERE organization_id = org_id
  AND EXTRACT(YEAR FROM created_at) = EXTRACT(YEAR FROM NOW());

  new_number := 'COT-' || year_part || '-' || count_part;
  RETURN new_number;
END;
$$ LANGUAGE plpgsql;
```

```typescript
// lib/pdf/quote-generator.ts
import { jsPDF } from 'jspdf';
import autoTable from 'jspdf-autotable';

export async function generateQuotePDF(quote: Quote) {
  const doc = new jsPDF();

  // Logo (si existe)
  if (quote.organization.logo_url) {
    doc.addImage(quote.organization.logo_url, 'PNG', 10, 10, 30, 30);
  }

  // Header
  doc.setFontSize(20);
  doc.text('COTIZACIÓN', 150, 20);
  doc.setFontSize(10);
  doc.text(`Folio: ${quote.quote_number}`, 150, 30);
  doc.text(`Fecha: ${formatDate(quote.created_at)}`, 150, 35);
  doc.text(`Vigencia: ${formatDate(quote.valid_until)}`, 150, 40);

  // Datos técnico
  doc.setFontSize(12);
  doc.text(quote.organization.name, 10, 50);
  doc.setFontSize(10);
  doc.text(`Tel: ${quote.organization.phone}`, 10, 55);
  doc.text(`Email: ${quote.organization.email}`, 10, 60);

  // Datos cliente
  doc.setFontSize(12);
  doc.text('CLIENTE:', 10, 75);
  doc.setFontSize(10);
  doc.text(quote.customer.name, 10, 80);
  doc.text(`Tel: ${quote.customer.phone}`, 10, 85);
  if (quote.customer.address) {
    doc.text(quote.customer.address, 10, 90);
  }

  // Tabla de items
  autoTable(doc, {
    startY: 100,
    head: [['Descripción', 'Cant.', 'P. Unit.', 'Total']],
    body: quote.items.map(item => [
      item.description,
      item.quantity.toString(),
      formatCurrency(item.unit_price),
      formatCurrency(item.total)
    ]),
    foot: [
      ['', '', 'Subtotal:', formatCurrency(quote.subtotal)],
      ['', '', 'IVA (16%):', formatCurrency(quote.tax)],
      ['', '', 'Descuento:', formatCurrency(quote.discount)],
      ['', '', 'TOTAL:', formatCurrency(quote.total)]
    ],
    theme: 'striped',
    headStyles: { fillColor: [41, 128, 185] },
    footStyles: { fillColor: [236, 240, 241], textColor: [0, 0, 0] }
  });

  // Términos
  const finalY = (doc as any).lastAutoTable.finalY + 10;
  doc.setFontSize(8);
  doc.text('Términos y Condiciones:', 10, finalY);
  doc.text('1. Precios en pesos mexicanos + IVA', 10, finalY + 5);
  doc.text('2. Validez de 15 días naturales', 10, finalY + 10);
  doc.text('3. Anticipo 50% para iniciar trabajos', 10, finalY + 15);
  doc.text('4. Garantía 1 año en instalación', 10, finalY + 20);

  return doc.output('blob');
}
```

**Código Ejemplo - Cron Recordatorios**:
```typescript
// app/api/cron/send-reminders/route.ts
import { NextResponse } from 'next/server';
import { createServerClient } from '@/lib/supabase/server';
import { sendWhatsAppMessage } from '@/lib/whatsapp';

export async function GET(request: Request) {
  // Seguridad: verificar cron secret
  const authHeader = request.headers.get('authorization');
  if (authHeader !== `Bearer ${process.env.CRON_SECRET}`) {
    return NextResponse.json({ error: 'Unauthorized' }, { status: 401 });
  }

  const supabase = await createServerClient();
  const today = new Date().toISOString().split('T')[0];

  // Buscar recordatorios para hoy
  const { data: reminders } = await supabase
    .from('automated_reminders')
    .select(`
      *,
      customers (*),
      organizations (*)
    `)
    .eq('next_reminder_date', today)
    .eq('is_active', true);

  let sent = 0;
  let errors = 0;

  for (const reminder of reminders || []) {
    try {
      // Reemplazar variables en template
      const message = reminder.message_template
        .replace('{nombre}', reminder.customers.name)
        .replace('{negocio}', reminder.organizations.name);

      // Enviar WhatsApp
      await sendWhatsAppMessage(
        reminder.customers.phone,
        message
      );

      // Actualizar siguiente fecha
      const nextDate = new Date(reminder.next_reminder_date);
      nextDate.setMonth(nextDate.getMonth() + reminder.frequency_months);

      await supabase
        .from('automated_reminders')
        .update({
          next_reminder_date: nextDate.toISOString().split('T')[0],
          last_sent_at: new Date().toISOString()
        })
        .eq('id', reminder.id);

      sent++;
    } catch (error) {
      console.error(`Error sending reminder ${reminder.id}:`, error);
      errors++;
    }
  }

  return NextResponse.json({
    success: true,
    sent,
    errors,
    timestamp: new Date().toISOString()
  });
}
```

```typescript
// lib/whatsapp/index.ts
import twilio from 'twilio';

const client = twilio(
  process.env.TWILIO_ACCOUNT_SID,
  process.env.TWILIO_AUTH_TOKEN
);

export async function sendWhatsAppMessage(
  to: string,
  message: string,
  mediaUrl?: string
) {
  try {
    const result = await client.messages.create({
      from: `whatsapp:${process.env.TWILIO_WHATSAPP_NUMBER}`,
      to: `whatsapp:+52${to}`,
      body: message,
      ...(mediaUrl && { mediaUrl: [mediaUrl] })
    });

    return { success: true, sid: result.sid };
  } catch (error) {
    console.error('WhatsApp error:', error);
    return { success: false, error };
  }
}

export async function sendQuoteViaWhatsApp(quoteId: string) {
  const supabase = await createServerClient();

  const { data: quote } = await supabase
    .from('quotes')
    .select('*, customers(*), organizations(*)')
    .eq('id', quoteId)
    .single();

  if (!quote) throw new Error('Quote not found');

  const message = `
Hola ${quote.customers.name}! 👋

Te envío la cotización de ${quote.organizations.name}:

📄 Folio: ${quote.quote_number}
💰 Total: ${formatCurrency(quote.total)}
📅 Válida hasta: ${formatDate(quote.valid_until)}

Para ver el detalle completo, descarga el PDF adjunto.

¿Te interesa? Responde SÍ para agendar el servicio.

Gracias! 🙌
  `.trim();

  const result = await sendWhatsAppMessage(
    quote.customers.phone,
    message,
    quote.pdf_url // PDF adjunto
  );

  if (result.success) {
    // Marcar como enviada
    await supabase
      .from('quotes')
      .update({
        status: 'sent',
        sent_at: new Date().toISOString()
      })
      .eq('id', quoteId);
  }

  return result;
}
```

```typescript
CREATE TABLE referrals (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  referrer_id UUID NOT NULL REFERENCES profiles(id),
  referee_email TEXT NOT NULL,
  referee_id UUID REFERENCES profiles(id),
  referral_code TEXT UNIQUE NOT NULL,
  status TEXT DEFAULT 'pending' CHECK (status IN ('pending', 'signed_up', 'paid', 'rewarded')),
  signed_up_at TIMESTAMPTZ,
  first_payment_at TIMESTAMPTZ,
  reward_granted_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_referrals_code ON referrals(referral_code);
CREATE INDEX idx_referrals_referrer ON referrals(referrer_id);

-- Función para generar código de referido
CREATE OR REPLACE FUNCTION generate_referral_code(user_id UUID)
RETURNS TEXT AS $$
DECLARE
  code TEXT;
BEGIN
  -- Generar código único de 8 caracteres
  code := UPPER(SUBSTRING(MD5(user_id::TEXT || NOW()::TEXT) FROM 1 FOR 8));
  RETURN code;
END;
$$ LANGUAGE plpgsql;
```

## 11. Riesgos y Mitigaciones

### Riesgo 1: Baja Adopción Tecnológica 🔴

**Probabilidad**: Alta (60%)
**Impacto**: Alto

**Descripción**: Técnicos mayores (45+ años) resistentes a adoptar nueva tecnología.

**Mitigaciones**:
```
✅ Onboarding 1-a-1 obligatorio (no self-service)
✅ Videos tutoriales MUY simples (paso a paso)
✅ Soporte por WhatsApp (su canal preferido)
✅ Interfaz ultra simple:
   - Botones grandes
   - Textos claros
   - Sin jerga técnica
   - Flujos de 2-3 pasos máximo
✅ Freemium generoso (prueban sin riesgo)
✅ Contratar vendedor con experiencia en sector HVAC
   (habla su idioma, entiende su negocio)
```

**Plan B**: Si adopción < 30%, pivotear a target más joven (25-35 años) o verticalizarse en instaladores empresariales.

---

### Riesgo 2: WhatsApp Business API Rechazado 🔴

**Probabilidad**: Media (30%)
**Impacto**: Crítico

**Descripción**: Twilio/Meta rechazan solicitud de WhatsApp Business por "uso masivo de mensajería".

**Mitigaciones**:
```
✅ Aplicar con caso de uso B2B claro (no marketing masivo)
✅ Demostrar opt-in de usuarios (solo clientes que aceptaron)
✅ Rate limits conservadores (max 500 msg/día por técnico)
✅ Plan B: SMS tradicional
   - Más caro ($0.50/SMS vs $0.10/WhatsApp)
   - Pero 100% aprobación
   - Pasar costo a usuario ($0.75/SMS)
✅ Plan C: Integración WhatsApp Web (no oficial)
   - Usar Baileys library (Node.js)
   - Riesgoso: pueden banear cuenta
   - Solo para beta/validación
✅ NO prometer WhatsApp en ventas hasta tener aprobación
```

**Contingencia**: Si rechazan en mes 2, lanzar con SMS. Agregar WhatsApp en V2 cuando tengamos tracción (más fácil aprobar con uso real).

---

### Riesgo 3: Churn Alto Primeros 3 Meses 🟡

**Probabilidad**: Alta (70%)
**Impacto**: Alto

**Descripción**: 40-50% de usuarios cancelan en primeros 90 días por no ver valor.

**Mitigaciones**:
```
✅ Onboarding perfecto (80% churn es por mala onboarding)
   - Asegurar que envían 1era cotización en onboarding call
   - No terminar llamada hasta que entiendan 100%

✅ Check-ins proactivos:
   - Día 3: WhatsApp "¿Cómo vas?"
   - Día 7: Llamada de follow-up
   - Día 14: Email con tips avanzados
   - Día 21: Oferta de renovación anticipada (descuento)
   - Día 28: Último push antes de renovación

✅ Feature "Time to Value" < 1 hora
   - Primera cotización enviada en < 60 min
   - Primer cliente guardado en < 30 min
   - Primer recordatorio programado en < 15 min

✅ Gamificación:
   - Badges por logros:
     • "Primera Cotización Enviada"
     • "10 Clientes Guardados"
     • "Primera Venta Cerrada"
   - Progress bar de setup

✅ Contratos anuales con descuento:
   - Mensual: $199/mes
   - Anual: $1,990/año (ahorro $400)
   - Pre-pago reduce churn 60-70%

✅ Exit interviews:
   - Entender por qué cancelan
   - Ofrecer pausa en vez de cancelar
   - Win-back campaign después de 3 meses
```

**Target Churn**:
- Mes 1-3: < 20%
- Mes 4-6: < 12%
- Mes 7+: < 8%

**Probabilidad**: Media (40%)
**Impacto**: Medio

**Descripción**: Competidor grande (ServiceTitan, Jobber) lanza versión en español o startup mexicano copia.

**Mitigaciones**:
```
✅ Velocidad de ejecución:
   - MVP en 6 semanas (antes que copien)
   - Launch en 3 meses
   - Iteración semanal

✅ Moats defensibles:
   - Relaciones con distribuidores (exclusividad 1 año)
   - Data de clientes (switching cost alto)
   - Integración con catálogos mexicanos
   - Soporte en español (no traducción, nativo)

✅ Nicho ultra-específico:
   - Solo México (entendemos SAT, CFDI, cultura)
   - Solo técnicos HVAC (profundidad > amplitud)
   - Features específicos (ej: integración con distribuidores locales)

✅ Comunidad:
   - Grupo privado Facebook de usuarios
   - Eventos trimestrales (virtual)
   - Programa de embajadores

✅ Innovación continua:
   - 1 feature nueva/mes
   - Roadmap público
   - Beta access a power users
```

**Si competidor lanza**:
- Analizar qué hacen diferente
- Doblar down en servicio al cliente
- Resaltar diferenciadores en marketing
- Ofrecer migración gratis de competidor

### Riesgo 5: Costos WhatsApp/Twilio Insostenibles 🟡

**Probabilidad**: Media (35%)
**Impacto**: Alto

**Descripción**: Usuarios envían 2,000+ WhatsApp/mes, COGS explotan.

**Mitigaciones**:
```
✅ Límites claros por plan:
   - Free: 0 WhatsApp
   - Pro: 500 WhatsApp/mes
   - Ultra: 2,000 WhatsApp/mes
   - Overage: $0.50 por cada 10 mensajes extra

✅ Optimización de envíos:
   - Batch reminders (1x/día no 1x/hora)
   - Consolidar notificaciones
   - Smart throttling (no enviar 2am-7am)

✅ Monitoreo COGS semanal:
   - Dashboard con costo/usuario
   - Alertas si usuario > $20/mes COGS
   - Análisis de power users

✅ Renegociación con Twilio:
   - Al llegar a 100K msg/mes, negociar descuento
   - Volumen = poder de negociación
   - Target: $0.08/msg (vs $0.10 actual)

✅ Plan B: Modelo de créditos
   - $199/mes incluye 500 créditos
   - 1 crédito = 1 WhatsApp
   - Créditos extra: $50 por 100 créditos
   - Usuarios conscientes de uso

✅ Email como fallback gratis:
   - Si no hay créditos, envía email
   - Email es gratis (SendGrid 10K/mes free)
```

**Break-even COGS**:
- ARPU target: $250/mes
- COGS target: < 10% ($25/mes)
- Margen objetivo: 90%+

```
   - ¿Pricing muy alto?
   - ¿Target equivocado?

✅ Pivotes posibles:

   Pivote 1: Cambiar target
   - De técnicos individuales → empresas HVAC
   - De minisplit → plomería/electricidad
   - De México → USA (mercado más maduro)

   Pivote 2: Cambiar pricing
   - De $199/mes → $99/mes
   - De suscripción → pay-per-quote ($5/cotización)
   - De mensual → anual prepago (descuento 50%)

   Pivote 3: Cambiar propuesta de valor
   - De "genera cotizaciones" → "cierra más ventas"

   Pivote 4: Cambiar a otra idea

✅ Contingencia: Desarrollo custom

   - Validas demanda
   - Generas cash para runway
   - Aprendes caso de uso real
   - Después conviertes en SaaS
```
